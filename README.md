# React Tutorial Repository 🚀

Welcome to the **React Tutorial Repository**! This collection of projects is designed to help developers—from beginners to advanced—learn, master, and practice React development. Each project inside this repository is a standalone application focusing on different concepts and modern web development practices.

## 📂 Projects Included

This repository contains the following comprehensive tutorials and starter templates:

### 1. [To-Do List Tutorial](./do_to_list)
A beautifully designed, beginner-friendly web application built with React and Vite. It serves as both a fully functional To-Do list and an interactive tutorial for learning React basics.

**Key Features:**
- **Interactive Tutorial**: Step-by-step guidance for beginners.
- **Modern UI/UX**: Features a sleek dark mode with frosted glassmorphism effects and smooth micro-animations.
- **Fully Functional Demo**: A built-in working To-Do list to demonstrate the final product.
- **Tech Stack**: React, Vite, Vanilla CSS, Lucide React (for SVG icons).

### 2. [Basic eCommerce](./Basic_eCommerce)
A foundational template providing a minimal setup to get React working seamlessly in Vite. This project is the perfect starting point for building more complex, production-ready applications like an eCommerce store.

**Key Features:**
- **Vite & HMR**: Lightning-fast hot module replacement.
- **Oxlint Setup**: Pre-configured linting rules for writing clean code.
- **Scalable Architecture**: Ready to be expanded into a full-fledged eCommerce application.
- **Tech Stack**: React, Vite, Oxc / SWC compilation.

## 💻 How to Run Locally

If you'd like to run any of these projects on your own machine, follow these standard steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Setup Instructions

1. **Clone the repository** (or navigate into the specific project folder you want to run):
   ```bash
   # Example: navigating to the To-Do List project
   cd do_to_list
   ```

2. **Install Dependencies**:
   Install all the required NPM packages for the project.
   ```bash
   npm install
   ```

3. **Start the Development Server**:
   Run the local Vite development server.
   ```bash
   npm run dev
   ```

4. **View the Application**:
   Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal).
